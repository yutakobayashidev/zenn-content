WordPress の改ページ( `<!--nextpage-->` または「ページ区切り」ブロック )でページごとにデザインを変更しようとした時のメモです。

# グローバル変数を使えばできる

結論から言うとこれだけでできます。
`$page` は投稿ページや固定ページのページ番号を取得できるグローバル変数です。

このコードの場合は「1 ページにのみ何かを表示する」といった感じです。

```php
<?php  if ($page < 2): ?>
ここに表示させたいものを入力
<?php  endif; ?>
```

また `$paged` という似たようなグローバル変数もありますがこちらはカテゴリーページやタグページ用なので注意してください。
